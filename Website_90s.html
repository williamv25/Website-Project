BASIC(1964)

BASIC is one of earliest programming languages published in 1964. It was developed by John G. Kemeny, Thomas E. Kurtz and Mary Kenneth Keller at Dartmouth College. BASIC stands for "Beginner's All-purpose Symbolic Instruction Code." It was designed to allow students at Dartmouth College to write mainframe computer programs for their Time-Sharing System. It concentrated on mathematical work containing arithmetic functions inside of the program.

PASCAL(1970)

PASCAL is a procedural programming language designed by Niklaus Wirth that was published in 1970. It was based on a concept called Structure Programming that had become recently popular. It contained data structures such as lists and graphs.
